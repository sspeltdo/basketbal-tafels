# Basketbal Tafels 🏀

Een basketbal-thema oefenapp voor vermenigvuldigen en delen — gemaakt voor jonge leerlingen.

## Wat doet de app?

- Kies welke tafels je wilt oefenen: vermenigvuldigen (× 1 t/m × 10) en/of delen (÷ 2 t/m ÷ 10)
- Stel het aantal vragen in (5–20)
- Beantwoord vragen via meerkeuze of invulveld
- Krijg direct feedback met animaties
- Zie je eindscore met een aanmoedigend bericht

## GitHub Pages inschakelen

1. Ga naar je repository op GitHub
2. Klik op **Settings** (instellingen)
3. Scroll naar **Pages** in het linkermenu
4. Kies onder **Source** de branch `main` en de map `/ (root)`
5. Klik op **Save**
6. Na een minuutje is je app live op:

```
https://sspeltdo.github.io/basketbal-tafels/
```

## Technisch

- Pure HTML + CSS + JavaScript (geen frameworks, geen backend)
- Werkt op desktop en mobiel
- Taal: Nederlands 🇧🇪🇳🇱
